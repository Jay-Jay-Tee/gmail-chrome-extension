How to get a Gemini API key
  1. Open Google AI Studio – API Keys.
  2. Sign in with any Google account.
  3. Click Create API key and copy it.
  4. Paste it below and click Save.
  5. Click Verify key works to complete setup.

API error 4xx/5xx → key is invalid or rate-limited. Create a new one.

Unlocks the Inbox Dashboard: connected email display, stats, and one-click inbox actions. You can skip this and connect later from the extension popup.

How to set up OAuth
  1. Create a project in Google Cloud Console.
  2. Enable Gmail API for that project.
  3. Set up the OAuth consent screen (External, Testing mode, add your Gmail as test user).
  4. Create an OAuth client ID for Chrome Extension.
  5. Paste your extension ID from chrome://extensions as the Item ID.
  6. Copy the OAuth Client ID into manifest.json under oauth2.client_id.
  7. Reload the extension, then click Connect Gmail below.
